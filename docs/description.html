# ⚔️ RGX | Skyrim Replacement Level-Up! ## ![](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Skyrim Replacement Level-Up!** brings the iconic Skyrim level-up chime to World of Warcraft, replacing the default sound with that triumphant, legendary fanfare — across all WoW versions. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) **The Kiwi Says:** "Fus Ro DAH! Bwwiiiee." *** ## ✨ Features 🎵 Epic Sound - The authentic Skyrim level-up chime plays every time you gain a level! 🎚️ Quality Options - Choose between High, Medium, and Low quality sound variants to suit your preference. 🔇 Smart Muting - Auto-mutes WoW's default level-up sound so you never hear it again! 🌍 Multi-Language - Supports English, German, French, Spanish, and Russian with automatic detection. 💾 Persistent Settings - Your settings are saved automatically between sessions. ⚡ Optimized Performance - Cached constants and efficient code keep it lightweight. 🛡️ Error Protected - pcall error handling throughout for maximum stability. 🎮 Multi-Version - Works across all WoW versions — Classic Era, Cataclysm, Mists, and Retail! **Note:** All quality variants are included — no additional downloads required. *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/srlu help # Show all available commands
/srlu test # Play test sound
/srlu enable # Enable the addon
/srlu disable # Disable the addon
/srlu high # Use high quality sound
/srlu med # Use medium quality sound (default)
/srlu low # Use low quality sound
